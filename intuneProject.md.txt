# Intune Endpoint Management & Conditional Access Project

## Project Overview
This project simulates **real-world IT Support / System Administrator tasks** using Microsoft Intune, Entra ID, and Azure. The goal is to demonstrate how to onboard users, manage devices, deploy applications, enforce compliance, and implement security policies in a corporate environment.

**Key Objectives:**
- Onboard a new employee (Navya Sruthi) in Microsoft 365.
- Enroll devices into Intune for management.
- Configure compliance and security policies.
- Deploy Microsoft 365 Apps (Teams, Outlook, Word, Excel).
- Create Conditional Access policies for secure access.
